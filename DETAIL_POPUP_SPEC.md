# 大弹窗规范 v1

适用于家装方案详情、直播间详情等 PC 大弹窗。

## 容器
- 弹窗尺寸：1776 x 1032。
- 背景：`#FFF5D8`，使用 `assets/img/detail-bg-pattern.jpg`，`background-size: cover`。
- 圆角：顶部 18px，底部 16px。
- 阴影：`0 28px 72px rgba(0, 0, 0, .42)`。
- 蒙层：60% 黑，弹窗内局部菜单蒙层约 38% 黑。

## 字号
- 大弹窗主标题：`var(--fs-mega)`，48px，标题字体。
- 分享标题：48px，颜色 `#FDF3E0`。
- 主要按钮：48px，标题字体。
- 作者昵称大展示：36px 或 48px，按空间定。
- 卡片/评论主体强调文字：26px。
- 普通评论元信息、回复、作者赞过：21px。
- 不使用低于 21px 的字号。

## 左侧封面作者卡
- 封面沿用入口卡片视觉体系。
- 玩法标签：米黄底、棕色描边、小圆角。
- 精选标签：使用 `tag-featured.png` 切图，允许浮出，不裁切。
- 头像无阴影。
- 关注/已关注按钮等宽等高。

## 评论区
- 一级评论列表不展示“回复某人”的评论样式。
- 二级评论详情第一条评论下方无分割线，直接接“共 xx 条回复”组件。
- 评论区原生滚动条隐藏，使用自定义无箭头竖条。
- 输入态默认一行，超一行向上增高，最多四行；超过四行内部滚动。
- 超过 40 字允许输入，但计数变红；发送时 toast 提示。

## 底栏
- 左半区放数据，右半区放操作按钮。
- 数据区：已游玩 / 收藏 / 点赞。
- 收藏、点赞：图标跨两行居中，右侧上行 label，下行数字。
- 操作按钮圆角 28px，与“我的”个人卡片大按钮一致。

## 局部菜单气泡
- 宽度：300px。
- 字号：40px。
- 两项竖排，中间横线。
- 右上更多：删除 / 编辑。
- 评论三点：删除 / 举报评论。
